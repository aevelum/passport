# 02 Foundation Release Scope

The Foundation Release proves one committee-ready workflow:

> repo pre-trade collateral-capacity credentialing.

## In scope

- Daml project.
- Core templates.
- Repo pre-trade demo.
- Daml Script tests.
- Privacy assertions.
- CDM-aligned mapping draft.
- Local structural gates.

## Out of scope

- Live custodian integration.
- Live venue integration.
- Live settlement.
- Custody.
- Collateral movement.
- Collateral optimization.
- Credit approval.
- ZK proofs.
- Diligence workspace UI.
- Production identity integration.
- Formal CDM conformance.

## Acceptance definition

The Foundation Release is acceptable when it can show:

1. policy publication;
2. credential request;
3. credential issuance;
4. scoped presentation;
5. partial reservation;
6. residual credential issuance;
7. reservation release;
8. revocation;
9. scoped audit disclosure;
10. unauthorized party non-visibility.
