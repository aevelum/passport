# 02 Foundation Release Scope

The Foundation Release proves one committee-ready workflow:

> repo pre-trade collateral-capacity credentialing.

## In scope

- Daml project.
- Core templates.
- Repo pre-trade demo.
- Daml Script tests.
- Privacy assertions.
- Framework-neutral interop adapter surface.
- CDM 6.0 collateral eligibility plugin.
- Local structural gates.

## Out of scope

- Not live custodian integration.
- Not live venue integration.
- Not live settlement.
- Not custody.
- Not collateral movement.
- Not collateral optimization.
- Not credit approval.
- Not ZK proofs.
- Not diligence workspace UI.
- Not production identity integration.
- Not Rosetta Engine execution.
- Not formal standards-body endorsement.

## Acceptance definition

The Foundation Release is acceptable when it can show:

1. policy publication;
2. credential request;
3. credential issuance;
4. scoped presentation;
5. partial reservation;
6. residual credential issuance;
7. reservation release;
8. revocation;
9. scoped audit disclosure;
10. unauthorized party non-visibility.
